# Отстранување дупликати од Excel табела

Скрипта за чистење Excel табела:
- **Наоѓа дупликати** по комбинација Ime + Prezime (или друга колона)
- **Задржува го редот со повеќе пополнети колони** (повеќе податоци за лицето)
- **Го брише редот со помалку податоци**
- **Сортира по датум**

## Инсталација

```bash
pip install pandas openpyxl
```

## Употреба

```bash
python3 deduplicate.py --input vashiot_fajl.xlsx --output rezultat.xlsx
```

Скриптата **автоматски ги открива** колоните:
- Ако постојат одделни `Ime` и `Prezime` колони → ги зема двете
- Ако постои само `Ime i Prezime` → ја зема неа
- Датумот исто така се открива автоматски

### Рачно наведување колони

```bash
# Две одделни колони: Ime и Prezime
python3 deduplicate.py \
    --input lista.xlsx --output rezultat.xlsx \
    --name-cols "Ime" "Prezime"

# Една комбинирана колона
python3 deduplicate.py \
    --input lista.xlsx --output rezultat.xlsx \
    --name-cols "Ime i Prezime" \
    --date-col "Datum na vrabotuvanje"
```

### Сите опции

| Аргумент | Опис | Default |
|----------|------|---------|
| `--input` | Влезен Excel фајл | (задолжително) |
| `--output` | Излезен Excel фајл | `deduplicated_<input>` |
| `--name-cols` | Колони со имиња (може повеќе) | auto-detect |
| `--date-col` | Колона со датуми | auto-detect |
| `--sheet` | Sheet (број или ime) | 0 (прв sheet) |
